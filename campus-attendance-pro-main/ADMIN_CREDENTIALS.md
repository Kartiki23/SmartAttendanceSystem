## 🔧 Useful Commands

### Create Admin User
```bash
cd backend
node scripts/createAdmin.js
```

### Seed Database with Classes & Subjects
```bash
cd backend
node scripts/seedData.js
```

### Start Backend Server
```bash
cd backend
npm run dev
```

### Start Frontend Server
```bash
npm run dev
```
